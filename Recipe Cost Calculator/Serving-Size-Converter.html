Serving Size Converter Documentation

Introduction

The Serving Size Converter is a tool designed to help users adjust ingredient quantities for different serving sizes. This is especially useful when scaling recipes up or down to fit the number of guests or servings needed.

Features

  • Convert ingredient quantities between different serving sizes.
  • Add and remove ingredients dynamically.
  • Download the converted ingredient list as a PDF.
  • Responsive and accessible design with a dark theme.

How to Use

  1. Input Original Servings: Enter the number of servings for the original recipe in the "Original Number of Servings" field.
  2. Input New Servings: Enter the desired number of servings for the adjusted recipe in the "New Number of Servings" field.
  3. Add Ingredients: Click the "Add Ingredient" button to add ingredients to the list. For each ingredient, provide the name, quantity, and unit (e.g., cups, grams).
  4. Convert Quantities: Click the "Convert" button to adjust the ingredient quantities based on the new number of servings.
  5. Download PDF: Click the "Download PDF" button to save the converted ingredient list as a PDF.

Example

If a recipe originally serves 4 people and you want to adjust it to serve 8 people, you would:

  • Enter "4" in the "Original Number of Servings" field.
  • Enter "8" in the "New Number of Servings" field.
  • Add the ingredients with their quantities and units for the original recipe.
  • Click "Convert" to get the adjusted quantities.
  • Download the PDF for easy reference.
